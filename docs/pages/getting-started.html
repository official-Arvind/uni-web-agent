Getting Started

Follow these instructions to run the Jigar Universal Web Agent locally.

1. Start the Backend Engine

Ensure your Python virtual environment is activated, then run the FastAPI server:

uvicorn backend.main:app --reload --port 8000

2. Start the UI Dashboard

Open a new terminal, navigate to the frontend directory, and run the Vite server:

cd frontend
npm run dev

3. Configure the Agent

Open your browser to http://localhost:5173.

  • Click ⚙️ Settings and enter your Gemini API Key.
  • Add a site using + Add New Site.
  • Generate your first AI-mapped workflow.
